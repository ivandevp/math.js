# Math.js

Esta librería cuenta con 4 métodos:

```javascript
import * as math from 'math';

math.suma(1, 2, 3, 4, 5, 6); // -> 21
math.resta(15, 4, 2); // -> 9
math.multiplicacion(3, 2, 1); // -> 6
math.division(20, 5); // -> 4
```

O lo puedes usar de esta forma:

```javascript
import {
  suma,
  resta,
  multiplicacion,
  division
} from 'math';

suma(1, 2, 3, 4, 5, 6); // -> 21
resta(15, 4, 2); // -> 9
multiplicacion(3, 2, 1); // -> 6
division(20, 5); // -> 4
```
